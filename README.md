# rapido-dashboard
The Rapido dashboard delivers key insights into completed rides, cancellations, peak times, and unique riders. Built with cleaned data and optimized DAX, it resolves issues like duplicate rider IDs and inconsistent time formats, providing clear, reliable performance reporting.
